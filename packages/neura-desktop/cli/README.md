# @neura-desktop/cli

CLI for Neura.

## Use adb operator in CLI

Connect an Android device to your computer and run the following command:

```bash
npx @neura-desktop/cli start -p 'your config' -t adb -q "Help me add Tom to my contacts. His phone number is 12345678900."
```
